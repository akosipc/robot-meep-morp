## Toy Robot Simulator


#### Version
```
  ruby 3.0.2
```

#### Installation and setup

```
  $ git clone git@github.com:akosipc/robot-meep-morp.git
  $ cd robot-meep-morp
  $ bundle install
```

#### Starting the application

```
  $ ./bin/start.rb
```

#### Running the test suite

```
  $ bundle exec rspec spec
```
